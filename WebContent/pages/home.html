DIVADIAWI

A Web-based Interface for Semi-automatic Labeling of Historical Document Images

DIVA group, University of Fribourg, Switzerland



The name DIVADIAWI stands for a Web-based Interface for Document Image Analysis of DIVA group (Document, Image and Voice Analysis group). DIVADIAWI was tested on IAM Historical Document Database (IAM-HistDB), which includes three datasets. Try the images from the database and their ground truth: Parzival dataset, Saint Gall dataset, and Washington dataset.
