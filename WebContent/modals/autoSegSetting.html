Set parameters for the automatic segmentation

The initial result obtained during the automatic segmentaion is a set of isolated words. We use some rectangles to link them together to form a text line. The width and height of the rectangle depend on the space between the two words. If the space is big, then the width of the rectangle should also be big. The height of the rectangle doesn't matter much. It should be adjusted only if the two words are not horizontal.
